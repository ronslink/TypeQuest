## Generation Instructions

### Option 1: ElevenLabs (Recommended)

```bash
# Set API key
export ELEVENLABS_API_KEY="your-key"

# Run generator
cd TypeQuest
node generate-tts.js --all

# Or for specific language
node generate-tts.js es
```

### Option 2: Edge TTS (Free, no key required)

```bash
# Install
npm install -g edge-tts

# Generate for one language
edge-tts -V en-US-AriaNeural -f prompts.txt -w output.mp3

# Use the prompts from above
```

### Option 3: Manual Recording

For best quality, consider professional voice actors for:
- English (primary narrator)
- Spanish (second largest market)
- French
- German

## Audio Specifications

| Setting | Value |
|---------|-------|
| Format | MP3, 192kbps |
| Sample Rate | 44.1 kHz |
| Duration | 1-3 seconds per sound |
| Volume | Normalized to -16 LUFS |
| Style | Encouraging, warm, patient |

## File Naming Convention

```
{Voice}/{Sound}.mp3
Examples:
  en/welcome.mp3
  es/countdown.mp3
  fr/encouragement.mp3
```
